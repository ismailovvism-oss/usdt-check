# Касса USDT (usdt-check)

Однофайловое веб-приложение для оператора обменника: за секунды отличить настоящий
USDT (TRC-20) от подделки, сверить сумму прихода, проверить операцию по хэшу, узнать
заморожен ли адрес в чёрном списке Tether и посчитать сумму к выдаче клиенту. Без
бэкенда — чистый клиент, данные берутся напрямую из TronGrid.

## Возможности
- **Приход** — последние входящие переводы на адрес, вердикт «Подлинно/Внимание».
- **Операция** — проверка перевода по хэшу транзакции или ссылке Tronscan.
- **Грязность** — проверка адреса по чёрному списку Tether (`isBlackListed`, on-chain).
- **Калькулятор** — расчёт суммы к выдаче (USDT⇄наличные, комиссия/наценка).
- **Радар** — следит за движением USDT между кошельками конкурентов и клиентов:
  ловит «клиент ушёл к конкуренту» и переводы конкурентов между собой. В отчёте
  вместо адресов — имена из твоей базы.
- **Мои кошельки** — до 5 своих адресов, сохраняются в браузере.
- **QR** — ввод адреса/хэша камерой или из картинки (встроенный `BarcodeDetector`).

## Критический инвариант
Настоящий USDT в TRON — единственный контракт `TR7NHqjeKQxGTCi8q8ZY4pL8otSzgjLj6t`.
Вердикт «Подлинно» возможен только если контракт совпал, получатель — это твой адрес
и (если задана) сумма совпала. Любое отклонение → «Внимание».

## Чего приложение не делает
Не проверяет происхождение монет (полный AML-таинт) и не гарантирует необратимость
перевода. Для крупных сумм всё равно открой кошелёк и убедись, что вырос расходуемый
баланс.

## Запуск
Открой `usdt-check.html` в браузере. Демо без сети: `usdt-check.html#demo=fake` /
`#demo=ok`. Свой адрес можно передать ссылкой: `usdt-check.html#addr=T…`.
